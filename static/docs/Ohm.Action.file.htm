val Ohm.Action.file

Returns a short, server-generated downloadable file back to the client. Typical usage and signature: 

<{ocaml:let response = Action.file ~mime ~file ~data response

val file : file:string -> mime:string -> data:string -> response -> response}>
  • file is the name of the file as it will appear on the user's filesystem after the download.
  • mime is the MIME type of the file.
  • data is a string representing the individual bytes of the file.

Do not use this function to return long or heavy files: you should instead upload those files to a file server (such a cloud hosting like Amazon S3, or maybe your own server), and give the user a download link.

Do not use this function to return static files, regardless of their size: use the ohm publish command-line command or the ohmStatic plug-in instead.

Example: sending a hello world text file.

<{ocaml:let hello req res = 
  let mime = "text/plain" in
  let file = "hello.txt" in
  let data = "Hello, world!" in
  return (Action.file ~file ~mime ~data res)}>
