type Ohm.Action.t

<{ocaml:type ('server,'args) t = 
      ('server,'args) request
   -> response
   -> (unit,response) Ohm.Run.t}>

An action is a function which explains how an HTTP request is turned into an HTTP response. It receives the HTTP request as its first parameter (as a request). It receives a default response as its second parameter, and must transform it into the actual response.

The action's result is wrapped in the asynchronous processing monad defined by module Ohm.Run.

The request contains the server instance (of type 'server) and the URL arguments (of type 'args).

Example: an action which expects a single URL argument of type string, and redirects the visitor to that URL.

<{ocaml:open Ohm
open Ohm.Universal 

let redirect req res =
  return (Action.redirect (req # args) res)}>
