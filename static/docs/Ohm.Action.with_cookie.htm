val Ohm.Action.with_cookie

Adds a cookie to the response. Leaves the original response unchanged, returns the modified response. Typical usage and signature: 

<{ocaml:let response = Action.with_cookie ~name ~value ~life response

val with_cookie : name:string -> value:string -> life:int -> response -> response}>
  • name is the cookie name.
  • value is the content of the cookie.
  • life is how long the cookie should remain valid, starting from the time when the response is received, in seconds. A value of zero means the cookie should remain valid until the browser is closed.

You may of course call this function multiple times to set several cookies in a single response.

Example: sending a session cookie.

<{ocaml:let with_session_cookie user_id res = 
  let name  = "SESSION" in
  let value = user_id in
  let life  = 3600 in
  Action.with_cookie ~name ~value ~life res)}>
