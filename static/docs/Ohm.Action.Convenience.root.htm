val Ohm.Action.Convenience.root

This function returns the root URL of a server instance. Typical usage and signature: 

<{ocaml:let url = Action.Convenience.root server instance

val root : 'i server -> 'i -> string}>

The generated URLs are of the form https://example.com:42 with no trailing slash — the port number is omitted if it is 443 for HTTPS or 80 for HTTP.
