val Ohm.Run.return

This function creates a run-box that always evaluates to the same value regardless of context. Typical use and signature: 

<{ocaml:let runbox = Run.return value 

val return : 't -> ('ctx,'t) t}>

This function is designed so that for any values ctx and x: 

<{ocaml:Run.eval ctx (Run.return x) == x}>

Ohm.Universal.return is an alias for this function.
