val Ohm.Action.redirect

Creates a 303 See Other HTTP redirect to the specified absolute URL. Typical usage and signature:

<{ocaml:let response = Action.redirect url response

val redirect : string -> response -> response}>

Any data previously bound to the response is discarded (HTML, JSON, JSONP, JS code, files...) with the exceptions of cookies, which are sent as normal.

Example: redirecting to this page. 

<{ocaml:Action.redirect "http://ohm-framework.com/docs/ohm.action.redirect" response}>
