All the introductory articles and tutorials for starting out with the Ohm Framework. Everyone was a newbie once.

Prerequisites

Ohm uses the OCaml language. You don't have to be an expert, and the tutorials will cover most of the complex parts anyway, but a minimal working knowledge of OCaml is recommended. If you can, take a little time to follow the Try OCaml interactive tutorial.

The OCaml Manual is a great reference, but not a good learning aid.

Extra credit: while you won't be forced to use them, Ohm works well with Coffeescript, jQuery, LESS CSS, CouchDB and the Batteries library for OCaml. As you become more experienced, you might want to read up on these as well.
